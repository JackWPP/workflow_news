# 高分子材料加工每日资讯平台 — 项目预算书

>
> 项目概述：基于 AI Agent 技术的高分子材料加工行业垂直资讯平台，实现多源新闻自动采集、智能筛选评估、结构化报告生成及交互式研究助手功能。

---

## 三、补充说明

1. **开发费用**涵盖从需求分析、系统设计、编码实现到部署上线的完整开发周期，同时包含开发期间产生的各项 API 调试费用。
2. **LLM 推理**采用 Kimi K2.5 模型（月度输入约 70M tokens、输出约 11M tokens），该模型支持自动上下文缓存机制，可在重复调用场景下降低约 80% 的输入成本。
3. **搜索引擎**采用 Brave Search 作为主力搜索源，该服务每月提供 $5 免费额度（约 1,000 次查询），超出部分按实际使用量计费。
4. **网页内容提取**采用 Firecrawl 与 Jina Reader 双链路架构，优先消耗免费额度，在单链路不可用时自动切换至备用链路，保障服务连续性。
5. **运维人力**主要覆盖日常自动化监控巡检、信源规则定期维护（RSS 订阅更新、搜索关键词调整）以及偶发的故障排查工作，按月均 2-3 小时人工介入估算。
6. 上述费用基于"每日自动生成 1 份行业报告"的标准负载测算，如后续报告频次或数据量显著增加，相关 API 费用将相应调整。
